# Ify & Kanayo Wedding Website

A beautiful, elegant wedding website for Ifeoma Nwaedozie and Kanayo Banor.

## 📅 Wedding Date
September 12, 2026

## 📍 Venue
Brick and Ivey, Marietta, GA, USA

## 🚀 Features

- **Interactive RSVP System** - Unique codes for guests, no plus-ones
- **Wedding Memories Gallery** - Share and view wedding photos
- **Gift Registry** - Amazon registry + cash donations (Zelle, PayPal, Stripe)
- **Wedding Details** - Ceremony, reception, schedule, practical info
- **Nearby Hotels** - Accommodation information for guests
- **Mobile Responsive** - Fully responsive design for all devices
- **Elegant Animations** - Smooth scroll effects and hover interactions

## 🛠️ Tech Stack

### Frontend
- React 19
- React Router DOM v7
- CSS Modules
- Vite
- React Icons

### Backend
- Node.js
- Express
- MySQL with Sequelize ORM
- Stripe Integration
- JWT Authentication (planned)
